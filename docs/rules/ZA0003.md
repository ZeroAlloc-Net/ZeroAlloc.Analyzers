# ZA0003: Use CollectionsMarshal.AsSpan for List iteration

## Description

Detects `foreach` over `List<T>` where `CollectionsMarshal.AsSpan()` can eliminate the enumerator allocation.

## Violation

```csharp
foreach (var item in list) { ... }
```

## Fix

```csharp
foreach (var item in CollectionsMarshal.AsSpan(list)) { ... }
```

## When to suppress

- The loop body contains `await` expressions (Span cannot cross await boundaries)
- The list may be modified during iteration

## Minimum TFM

net5.0

# ZA0004: Use SearchValues or FrozenSet for repeated lookups

## Description

Detects 4+ chained `||` equality comparisons against the same string variable. These can be replaced with `SearchValues<char>` or `FrozenSet<string>` for better performance.

## Violation

```csharp
if (s == "a" || s == "b" || s == "c" || s == "d") { ... }
```

## Fix

```csharp
private static readonly FrozenSet<string> _values =
    new[] { "a", "b", "c", "d" }.ToFrozenSet();

if (_values.Contains(s)) { ... }
```

## When to suppress

- The comparisons have side effects or short-circuit behavior that matters

## Minimum TFM

net8.0

# ZA0006: Use stackalloc for small fixed-size arrays

## Description

Detects `new byte[n]` or `new char[n]` where `n` is a constant ≤ 256 and the array is assigned to a local variable. `stackalloc` avoids heap allocation.

## Violation

```csharp
var buffer = new byte[128];
```

## Fix

```csharp
Span<byte> buffer = stackalloc byte[128];
```

## When to suppress

- The array escapes the method (returned, stored in a field, passed to async code)
- The method is already deeply recursive

## Minimum TFM

Any (requires C# 7.2+)

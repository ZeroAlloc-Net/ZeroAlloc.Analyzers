# ZA0008: Avoid Enum.HasFlag — use bitwise check

## Description

Detects `Enum.HasFlag()` calls on runtimes older than .NET 7, where `HasFlag` boxes the argument. On .NET 7+, the JIT inlines `HasFlag` so this diagnostic is suppressed.

## Violation

```csharp
if (options.HasFlag(Options.A)) { ... }
```

## Fix

```csharp
if ((options & Options.A) != 0) { ... }
```

## When to suppress

- Targeting .NET 7+ (diagnostic is automatically suppressed)
- Readability is preferred over performance in non-hot paths

## Minimum TFM

Reported on < net7.0 only

# ZA0010: Use TryGetValue instead of ContainsKey + indexer

## Description

Detects `if (dict.ContainsKey(key)) { ... dict[key] ... }` patterns where a single `TryGetValue` call avoids the double lookup.

## Violation

```csharp
if (dict.ContainsKey("key"))
{
    var value = dict["key"];
}
```

## Fix

```csharp
if (dict.TryGetValue("key", out var value))
{
    // use value
}
```

## When to suppress

- The dictionary is accessed concurrently and the double lookup is intentional for lock-free patterns

## Minimum TFM

Any

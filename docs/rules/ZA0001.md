# ZA0001: Use FrozenDictionary for read-only dictionary

## Description

Detects `readonly Dictionary<TKey, TValue>` fields that are never mutated after initialization. `FrozenDictionary` provides faster lookups for read-only data.

## Violation

```csharp
private readonly Dictionary<string, int> _lookup = new()
{
    ["a"] = 1, ["b"] = 2
};
```

## Fix

```csharp
private readonly FrozenDictionary<string, int> _lookup = new Dictionary<string, int>
{
    ["a"] = 1, ["b"] = 2
}.ToFrozenDictionary();
```

## When to suppress

- The dictionary is small (1-2 entries) and lookup speed is not critical
- The dictionary is intentionally mutable via reflection or unsafe code

## Minimum TFM

net8.0

# ZA0007: Use ArrayPool for large temporary arrays

## Description

Detects `new byte[n]` where `n` is a variable or constant > 256 in local scope. `ArrayPool<T>.Shared.Rent()` avoids large heap allocations.

## Violation

```csharp
var buffer = new byte[4096];
```

## Fix

```csharp
var buffer = ArrayPool<byte>.Shared.Rent(4096);
try
{
    // use buffer
}
finally
{
    ArrayPool<byte>.Shared.Return(buffer);
}
```

## When to suppress

- The array is long-lived and not temporary
- The array is passed to an API that requires exact length

## Minimum TFM

Any

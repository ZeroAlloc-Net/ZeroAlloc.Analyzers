# ZA0002: Use FrozenSet for read-only set

## Description

Detects `readonly HashSet<T>` fields that are never mutated after initialization. `FrozenSet` provides faster lookups for read-only data.

## Violation

```csharp
private readonly HashSet<string> _tags = new() { "a", "b", "c" };
```

## Fix

```csharp
private readonly FrozenSet<string> _tags = new HashSet<string> { "a", "b", "c" }.ToFrozenSet();
```

## When to suppress

- The set is small and lookup speed is not critical

## Minimum TFM

net8.0

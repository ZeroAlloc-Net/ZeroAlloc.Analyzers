# ZA0005: Avoid string concatenation in loops

## Description

Detects `+=` on a string variable inside a loop. Each concatenation allocates a new string. Use `StringBuilder` instead.

## Violation

```csharp
var result = "";
for (int i = 0; i < 100; i++)
    result += i.ToString();
```

## Fix

```csharp
var sb = new StringBuilder();
for (int i = 0; i < 100; i++)
    sb.Append(i);
var result = sb.ToString();
```

## When to suppress

- The loop is known to have very few iterations (< 3)

## Minimum TFM

Any

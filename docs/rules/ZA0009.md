# ZA0009: Avoid chained string.Replace calls

## Description

Detects 3+ chained `.Replace()` calls on a string. Each call allocates a new intermediate string. Use `StringBuilder` instead.

## Violation

```csharp
var result = s.Replace("a", "1").Replace("b", "2").Replace("c", "3");
```

## Fix

```csharp
var sb = new StringBuilder(s);
sb.Replace("a", "1");
sb.Replace("b", "2");
sb.Replace("c", "3");
var result = sb.ToString();
```

## When to suppress

- The string is short and the replacements are in non-hot code

## Minimum TFM

Any
